Professional Assessment

Throughout my time in the Computer Science program, I have developed a strong foundation in programming, software design, and problem solving. Completing this program and building my ePortfolio has allowed me to reflect on my growth as a developer while also giving me a way to showcase my technical skills to potential employers. The process of selecting, reviewing, and enhancing my artifacts helped me better understand not only how to write functional code, but how to improve, optimize, and present that code in a professional and meaningful way. One of the most valuable aspects of this program has been learning how to approach software development as a structured and collaborative process. While many of my projects were completed individually, I gained an understanding of how collaboration works in a professional environment, including writing clean, readable code that others can understand and maintain. I also developed skills in documenting my work and explaining technical concepts clearly, which is essential when communicating with stakeholders who may not have a technical background. Through assignments and project narratives, I practiced translating complex technical ideas into language that is more accessible, which is a skill I will continue to use in my career. My experience with data structures and algorithms has significantly improved my ability to design efficient and scalable solutions. Early in the program, my focus was primarily on getting code to work. However, as I progressed, I began to think more critically about performance, efficiency, and organization. For example, in my weight tracking mobile application, I implemented logic to evaluate user progress toward a goal weight and ensured that data was retrieved in an optimized manner using sorting techniques. This demonstrates my ability to apply algorithmic thinking to real-world problems and make informed decisions about trade-offs in design. In terms of software engineering and design, I have learned the importance of writing clean, maintainable, and well-structured code. While enhancing my mobile application, I refactored my code to separate concerns between the user interface and the database layer. This improvement not only made the application easier to manage and debug, but it also aligns with industry best practices. I also implemented proper error handling using try/catch blocks to ensure the application behaves reliably under unexpected conditions. These enhancements demonstrate my ability to evaluate existing code, identify weaknesses, and apply improvements that increase both functionality and maintainability. Database development has also been a key area of growth. I gained experience working with SQL and SQLite to design and manage data effectively. In my artifact, I implemented a structured database system that stores user information and weight tracking data. I enhanced this system by adding indexing to improve query performance and by validating user input to maintain data integrity. These improvements reflect my understanding of how databases support application functionality and how thoughtful design decisions can improve efficiency and reliability. Security is another important concept I have developed throughout the program. I have learned to think more critically about potential vulnerabilities and how to mitigate risks. In my mobile application, I incorporated permission checks before sending SMS messages, ensuring that the application follows proper security protocols. I also implemented input validation in my database methods to reduce the risk of invalid or harmful data being processed. While my understanding of security will continue to grow, I now have a solid foundation for recognizing and addressing common risks in software development. The artifacts included in my ePortfolio collectively demonstrate my growth across these areas. My weight tracking mobile application serves as a central piece that ties together software design, algorithms, and database functionality. By enhancing this application, I was able to showcase my ability to improve an existing system, apply best practices, and implement new features that add value. Each enhancement reflects a deeper level of understanding and a more professional approach to development. Overall, completing this program and developing my ePortfolio has prepared me to enter the field of computer science with confidence. I have not only gained technical skills, but I have also learned how to present those skills in a clear and professional manner. Moving forward, I plan to continue building on this foundation by expanding my knowledge, working on more complex projects, and pursuing opportunities that allow me to grow as a software developer. This ePortfolio represents both my current abilities and my potential for continued development in the field.
